Reactor Administration

Reactor is a load-balancing, scale management system for cloud application delivery built on top of Gridcentric's Virtual Memory Streaming technology. Reactor addresses the need for real-time provisioning of new instances according to changing load.

Reactor makes building truly reactive cloud services not only feasible but simple. Running as a virtual appliance in your VMS-enabled OpenStack cloud infrastructure, Reactor leverages the rapid virtual machine cloning technology of VMS to seamlessly scale services in response to load.
